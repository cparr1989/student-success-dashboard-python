# Student Success Dashboard

## Overview
A detailed Python dashboard that helps students track course performance, study hours, assignments, grades, and academic progress.

## Features
- Add multiple courses
- Track study hours
- Track completed assignments
- Track missing assignments
- Record quiz and test scores
- Add study notes
- Calculate assignment completion rate
- Calculate average quiz/test score
- Generate course performance status
- Generate overall academic summary

## Skills Demonstrated
- Python
- Functions
- Dictionaries
- Lists
- Loops
- Conditional Logic
- User Input
- Basic Data Analysis
- Dashboard Reporting
- Performance Tracking

## Why This Project Matters
This project demonstrates how Python can be used to organize information, analyze progress, and generate useful reports. It connects programming with real-world academic planning and data tracking.
